# ☕ Java Roadmap 2026 — от новичка до Senior

Полный практический roadmap по изучению и прокачке в Java: **теория + практика + пет-проекты**.
Каждый раздел содержит: *что учить → зачем → практику → как проверить себя*.

> **Как пользоваться:** иди сверху вниз. Не прыгай через уровни. На каждом этапе пиши код руками — без практики теория быстро забывается. Закрепляй каждый блок мини-проектом.

## Уровень 0. Подготовка и инструменты

**Цель:** настроить окружение и запустить первый код.

**Теория:**
- Что такое JDK, JRE, JVM и чем они отличаются.
- Как Java компилируется: \`.java\` → \`javac\` → байткод \`.class\` → JVM.
- Версии Java: учим на **LTS** (Java 17 или 21).

**Инструменты:**
- JDK 21 (LTS) — Temurin/Adoptium или Oracle.
- IDE: **IntelliJ IDEA Community** (рекомендуется).
- Git + аккаунт на GitHub.

**Практика:**
\`\`\`java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java 2026!");
    }
}
\`\`\`
Запусти из IDE и из терминала (\`javac Main.java && java Main\`).

**Проверь себя:** объясни словами путь от исходника до выполнения программы.

## Уровень 1. Основы языка (Core Java)

**Цель:** уверенно писать процедурный код.

**Теория:**
- Примитивные типы (\`int\`, \`long\`, \`double\`, \`boolean\`, \`char\`) и обёртки.
- Переменные, \`final\`, области видимости.
- Операторы, приведение типов.
- Условия: \`if/else\`, \`switch\` (в т.ч. switch-expression).
- Циклы: \`for\`, \`while\`, \`do-while\`, \`for-each\`.
- Массивы (одномерные и многомерные).
- Методы, перегрузка, \`varargs\`.
- Класс \`String\`, \`StringBuilder\`, форматирование.

**Практика:**
- FizzBuzz.
- Калькулятор в консоли.
- Реверс строки и проверка на палиндром.
- Поиск min/max и суммы в массиве.

**Проверь себя:** напиши программу, которая считает количество слов в строке.

## Уровень 2. ООП

**Цель:** мыслить объектами, а не функциями.

**Теория:**
- Класс и объект, поля и методы, конструкторы.
- 4 принципа: **инкапсуляция, наследование, полиморфизм, абстракция**.
- \`this\`, \`super\`, модификаторы доступа.
- \`static\` поля/методы, \`final\` классы.
- Абстрактные классы vs интерфейсы (\`default\`-методы).
- \`equals()\` / \`hashCode()\` / \`toString()\` — контракты.
- \`enum\`, records (Java 16+).
- Композиция против наследования.

**Практика:**
- Иерархия \`Shape\` → \`Circle\`, \`Rectangle\` с методом \`area()\`.
- Модель \`Bank\` с \`Account\`, переводы между счетами.
- Реализуй \`equals/hashCode\` руками и проверь в \`HashSet\`.

**Проверь себя:** объясни, почему "композиция лучше наследования".

## Уровень 3. Коллекции и Generics

**Цель:** выбирать правильную структуру данных.

**Теория:**
- Иерархия: \`Collection\`, \`List\`, \`Set\`, \`Queue\`, \`Map\`.
- \`ArrayList\` vs \`LinkedList\`, \`HashSet\` vs \`TreeSet\`.
- \`HashMap\` внутри: бакеты, хеши, коллизии.
- Generics: \`<T>\`, wildcards \`? extends\` / \`? super\`.
- \`Comparable\` vs \`Comparator\`.
- \`Iterator\`, \`Iterable\`.

**Практика:**
- Частотный словарь слов через \`HashMap\`.
- Сортировка списка объектов по нескольким полям.
- Реализуй свой простой \`Stack<T>\` на дженериках.

**Проверь себя:** когда \`LinkedList\` реально быстрее \`ArrayList\`?

---

## Уровень 4. Исключения, IO, функциональный стиль

**Цель:** надёжный код и современный синтаксис.

**Теория:**
- \`checked\` vs \`unchecked\` исключения, \`try/catch/finally\`, \`try-with-resources\`.
- Свои исключения.
- File IO: \`Files\`, \`Path\`, чтение/запись.
- Лямбды, функциональные интерфейсы (\`Function\`, \`Predicate\`, \`Supplier\`, \`Consumer\`).
- **Stream API**: \`map\`, \`filter\`, \`reduce\`, \`collect\`, \`Collectors\`.
- \`Optional\`.

**Практика:**
- Чтение CSV-файла и агрегация данных через Stream API.
- Обработка списка пользователей: фильтрация, группировка, сортировка.
- Свой класс исключения и корректная обработка.

**Проверь себя:** перепиши обычный цикл с фильтрацией на Stream API.

## Уровень 5. Многопоточность и JVM

**Цель:** понимать, как код исполняется и как работать параллельно.

**Теория:**
- Процессы и потоки, \`Thread\`, \`Runnable\`, \`Callable\`.
- \`synchronized\`, \`volatile\`, \`wait/notify\`.
- \`ExecutorService\`, пулы потоков, \`Future\`, \`CompletableFuture\`.
- Коллекции из \`java.util.concurrent\` (\`ConcurrentHashMap\`).
- Память JVM: heap, stack, GC (основы), метапространство.
- Race condition, deadlock.

**Практика:**
- Параллельная обработка данных через \`ExecutorService\`.
- Producer-Consumer на \`BlockingQueue\`.
- Замерь, как GC и пул потоков влияют на производительность.

**Проверь себя:** объясни, что такое deadlock, и приведи пример.

## Уровень 6. Базы данных и SQL

**Цель:** хранить и читать данные.

**Теория:**
- Реляционные БД (PostgreSQL / MySQL).
- SQL: \`SELECT\`, \`JOIN\`, \`GROUP BY\`, индексы, транзакции.
- JDBC: подключение, \`PreparedStatement\`.
- ORM-идея, **Hibernate / JPA**: сущности, связи, \`EntityManager\`.
- Connection pool (HikariCP).

**Практика:**
- CRUD через чистый JDBC.
- Тот же CRUD через JPA/Hibernate.
- Напиши запросы с JOIN и агрегацией.

**Проверь себя:** зачем нужен \`PreparedStatement\` вместо конкатенации строк? (SQL-инъекции!)

## Уровень 7. Сборка, тесты, Git

**Цель:** работать как в команде.

**Теория:**
- **Maven** и **Gradle**: зависимости, фазы сборки.
- Git: ветки, merge, rebase, pull request.
- Тестирование: **JUnit 5**, **Mockito**, AAA-паттерн.
- Покрытие тестами, юнит vs интеграционные тесты.

**Практика:**
- Настрой Maven/Gradle проект с зависимостями.
- Покрой бизнес-логику юнит-тестами.
- Замокай зависимость через Mockito.

**Проверь себя:** в чём разница между \`merge\` и \`rebase\`?

## Уровень 8. Spring / Spring Boot

**Цель:** строить реальные веб-приложения.

**Теория:**
- IoC и Dependency Injection, бины, контекст.
- Spring Boot: автоконфигурация, стартеры.
- REST: \`@RestController\`, \`@RequestMapping\`, статусы, DTO.
- Spring Data JPA: репозитории.
- Валидация, обработка ошибок (\`@ControllerAdvice\`).
- Конфигурация (\`application.yml\`), профили.
- Spring Security (основы), JWT.

**Практика:**
- REST API для задач (To-Do) с CRUD.
- Подключи БД через Spring Data JPA.
- Добавь авторизацию по JWT.

**Проверь себя:** что делает контейнер Spring при старте и что такое бин?

---

## Уровень 9. Backend в проде (продвинутый)

**Цель:** выйти на уровень Middle/Senior.

**Теория:**
- Архитектура: слои, чистая архитектура, DDD-основы.
- Микросервисы: REST vs gRPC, API Gateway, service discovery.
- Сообщения: **Kafka / RabbitMQ**.
- Кеширование: **Redis**.
- Docker и docker-compose, основы Kubernetes.
- CI/CD, мониторинг (Prometheus + Grafana), логирование.
- Паттерны проектирования (GoF), SOLID.
- Производительность: профилирование, оптимизация SQL, тюнинг JVM/GC.

**Практика:**
- Заверни сервис в Docker и подними с БД через docker-compose.
- Свяжи два сервиса через Kafka.
- Добавь кэш Redis и измерь ускорение.

**Проверь себя:** спроектируй простую систему (например, сокращатель ссылок) с нуля.
